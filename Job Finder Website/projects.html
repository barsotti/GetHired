Building a portfolio is crucial for getting hired in the technology industry. Completing these projects will not only show off your skills to possible employers, but more importantly help you learn more on the way. Below are some examples of projects that would allow you to showcase your abilities to possible employers.

Beginner

  • Turn your resume into a website
  • Create a set of icons

    • This will showcase your creativity and passion.

Intermediate

  • Create an interactive resume using javascript.

    • This project could actually go in beginner or advanced as well. It's up to you how in depth you go with this.

  • Design a program that determines the winner of a poker hand

    • A software developers job is to solve business problems, so projects that actually solve problems are great.

  • Find the website of a small or non profit company and modernize it

Advanced

  • Design a game using javascript.
  • Solve a business problem.
  • Completely redesign the website of your dream employer
  • Design a shell for a photo gallery that can be used by photographers or artists
  • Find a website who's mobile UX is lagging and modernize it
  • Contribute to open source projects on Github
